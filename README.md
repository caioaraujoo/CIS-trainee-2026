# Trainee CIS 2026 - Repositório de Atividades

Este repositório centraliza todas as minhas entregas e projetos práticos desenvolvidos durante o processo de Trainee da IEEE Computational Intelligence Society (CIS) na Universidade de Brasília.

## Sumário de Períodos

* **Período 1:** Aprendizado Supervisionado e Aprendizado Não Supervisionado.
* **Período 2:** Redes Neurais Artificiais e Classificação Estelar.

## Tecnologias Utilizadas
* Python
* Jupyter Notebook / VS Code
* PyTorch
