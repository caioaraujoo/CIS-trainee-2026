Como o banco de dados é muito pesado, ele não foi incluído neste repositório. Para rodar o código na sua máquina, siga os passos abaixo:

1. **Faça o download dos dados:** Baixe o arquivo CSV do dataset original no Kaggle, cujo link está listado logo abaixo de cada período.

2. **Organize as pastas:**
   Crie uma pasta chamada `data` na raiz deste projeto e coloque o arquivo `exemplo_banco_de_dados.csv` baixado dentro dela. A estrutura deve ficar exatamente assim:
   
   ```text
   meu-repositorio/
   │
   ├── data/                         <-- Crie esta pasta
   │   └── exemplo_banco_de_dados.csv   <-- Coloque o arquivo aqui
   │
   ├── período-2/
   │   └── exemplo.ipynb
   │
   ├── .gitignore
   └── README.md
